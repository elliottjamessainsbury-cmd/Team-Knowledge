# Beam Team Spaces

Beam Team Spaces is a front-end prototype showing how Beam Notes could become secure, source-grounded team memory for frontline teams.

The prototype uses local mock data only. It does not connect to real Beam systems, auth, APIs, databases, AI services, or real case data.

## Run locally

```bash
npm install
npm run dev
```

## What the prototype demonstrates

- Organisation-level workspace for Northbridge Council.
- Secure project/team spaces with documents, notes and collaborators.
- A polished right-side note canvas with summary, raw transcript, actions, comments and activity.
- Importing recent Beam Notes into a space using mocked sample data.
- Inviting collaborators with visible admin/audit framing.
- Supportive AI Space Brief, Ask this Space, source links and Combine Selected flow.
- Draft/source-grounded AI guardrails that require human review before action.

## Safety notes

- All data is synthetic.
- No real email, chat, SMS, auth, APIs or backend calls are used.
- AI outputs are deterministic mock content.
- AI does not make decisions, assess eligibility, provide legal advice or create statutory documents.
- Access changes are prototype-only and remain visible in the local audit panel.
